# Projeto Semestral: Gerenciador de Tarefas (TaskMaster)

## Descrição
Sistema de gerenciamento de tarefas pessoais desenvolvido em Java Swing. O projeto visa atender aos requisitos de avaliação semestral focando em organização, funcionalidade e robustez.

## Integrantes
- João Pedro Gonçalves da Silva

## Como Executar
1. Certifique-se de ter o Java (JDK 8+) instalado.
2. Clone este repositório.
3. Compile os arquivos na pasta `src`.
4. Execute a classe `com.projeto.Main`.

## Funcionalidades
1. Login (Admin/1234)
2. Listagem de Tarefas
3. Cadastro de Novas Tarefas
4. Edição/Exclusão e Conclusão
5. Dashboard de Estatísticas

## Performance e Testes
- Os testes unitários podem ser rodados via classe `TesteUnitario`.
- A análise de memória é exibida em tempo real na tela de Estatísticas.
